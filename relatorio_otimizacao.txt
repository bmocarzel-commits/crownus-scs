🔧 PRÓXIMOS PASSOS
════════════════════════════════════════════════════════════════════

1. ADICIONAR LAZY LOADING NO HTML

   Encontre estas linhas no seu index.html:

   <video id="rvid1" src="/reel1.mp4" loop playsinline></video>
   <video id="rvid2" src="/reel2.mp4" loop playsinline></video>
   ... e assim por diante ...

   MUDE PARA:

   <video id="rvid1" loading="lazy" src="/reel1.mp4" loop playsinline></video>
   <video id="rvid2" loading="lazy" src="/reel2.mp4" loop playsinline></video>

2. ADICIONAR IMAGEM WEBP

   Se você tem esta linha:
   
   <img src="/hairbooster.png" alt="..." class="hero-img" />

   MUDE PARA:

   <picture>
     <source srcset="/hairbooster.webp" type="image/webp">
     <img src="/hairbooster.png" alt="..." class="hero-img" />
   </picture>

3. FAZER UPLOAD DOS ARQUIVOS NO NETLIFY

   - Copie os vídeos comprimidos para seu projeto
   - Copie o arquivo hairbooster.webp
   - Atualize o HTML (passos 1-2 acima)
   - Faça git push ou arraste para Netlify UI

4. TESTAR COM LIGHTHOUSE

   Chrome DevTools > Lighthouse > Analyze page load
   Resultado esperado: Performance 85+ ✓
